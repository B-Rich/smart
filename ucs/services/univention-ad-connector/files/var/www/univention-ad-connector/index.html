Univention Active Directory Connector

The MSI files are the installation files for the password service and can be started by double clicking on it.

The package is installed in the C:\Windows\UCS-AD-Connector directory automatically. Additionally, the password service is integrated into the Windows environment as a system service, which means the service can be started automatically or manually.

ucs-ad-connector.msi Installation file for the password service for 32bit Windows.
It can be started by double clicking on it.
ucs-ad-connector-64bit.msi Installation file for the password service for 64bit Windows.
It can be started by double clicking on it.
vcredist_x86.exe Microsoft Visual C++ 2010 Redistributable Package (x86) - Must be installed on a 64bit Windows.
cert.pem The cert.pem file contains the SSL certificates created in UCS for secure communication. It must also be copied into the installation directory of the password service.
Please verify that the file has been downloaded as cert.pem, Internet Explorer appends a .txt under some circumstances.
private.key The private.key file contains the private key to the SSL certificates. It must also be copied into the installation directory of the password service.
