UCS Erstkonfiguration

Herzlich Willkommen

Die UCS Erstkonfiguration führt Sie durch die initiale Konfiguration, die notwendig ist, damit dieses neu erzeugte System verwendet werden kann.

Dieser Vorgang beinhaltet alle notwendigen Einstellungen, damit das System einer Domäne beitreten kann oder eine neue erzeugt wird. Darüber hinaus können weitere Einstellungen vorgenommen werden.


Die Erstkonfiguration des Systems ist aufgeteilt in die folgenden Schritte:

Während des Speichervorgangs wird auch der Domänenbeitritt durchgeführt. Dazu ist auf allen Systemrollen, außer Domänencontroller Master, die Eingabe von Benutzernamen und Passwort eines gültigen Domänenadministrator-Kontos notwendig. Auf dem Domänencontroller Master wird eine neue Domäne erzeugt. Nach dem erfolgreichen Domänenbeitritt wird der Einstellungsassistent beendet und der Boot-Vorgang fortgesetzt.

Werden die Einstellungen in einem Browser von einem externen System aus vorgenommen, ist im Anschluss an die Konfiguration eine Neuanmeldung an Univention Management Console erforderlich.

Die Erstkonfiguration kann jederzeit im UMC-Modul "Basiseinstellungen" geändert werden.
